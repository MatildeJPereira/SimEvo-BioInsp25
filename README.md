# Simulated Molecular Evolution

This project simulates the evolution of small molecules in a primordial-soup scenario using:

- Genetic Algorithms (GA)
- Novelty Search
- SELFIES molecular encoding
- RDKit (MMFF94 energies)

## Features

- SELFIES-based mutation and crossover
- MMFF94 stability scoring
- Tanimoto-based novelty search
- Population visualisation with RDKit
- Configurable μ + λ evolutionary scheme
